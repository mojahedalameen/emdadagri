**[1] تعريف الدور والمهمة**
أنت مصمم UI/UX خبير. صمم MVP لتطبيق "إمداد" بصلاحية "المزارع" فقط. المطلوب واجهات واضحة، عملية، غير فارغة، تركّز على الشراء السريع، طلب الاستشارة، وتتبع الطلبات. لا تستخدم صور أشخاص حقيقية أبداً، استخدم صور منتجات زراعية فقط، وأيقونات أو Illustrations للشخصيات. استخدم الأسماء الأولى فقط دون عوائل.

**[3] 🖥️ البنية العامة**
- Top Bar: شعار إمداد + إشعارات + اسم المستخدم "أهلاً، فهد"
- Bottom Navigation ثابت: الرئيسية / السوق / الاستشارات / طلباتي / حسابي
- اجعل كل شاشة مليئة بعناصر حقيقية، لا تترك فراغات كبيرة

**[4] 📄 الشاشات المطلوبة**

1. **الرئيسية**
- بطاقة ترحيب: "مرحباً فهد، ماذا تحتاج اليوم؟"
- شريط بحث
- 4 أزرار سريعة:
  - "اطلب الآن"
  - "استشارة فورية"
  - "تتبع شحنة"
  - "إعادة طلب"
- قسم فئات: بذور / أسمدة / مبيدات / معدات
- قسم منتجات مقترحة مع صور منتجات حقيقية
- قسم "آخر طلب" مع حالته

2. **السوق**
- شريط بحث
- فلترة أفقية: الكل / بذور / أسمدة / مبيدات / الأعلى تقييماً / الأقرب
- Grid منتجات: صورة، اسم المنتج، السعر، المورد، التقييم، حالة التوفر
- زر واضح في كل كرت: "أضف للسلة"

3. **تفاصيل المنتج**
- صورة منتج كبيرة
- الاسم، السعر، الوصف المختصر، المورد، التقييم، الكمية المتاحة
- اختيار الكمية + و -
- زران ثابتان أسفل الشاشة:
  - "أضف للسلة"
  - "استشر خبير عن هذا المنتج"

4. **الاستشارات**
- تبويبات: فورية / مجدولة / السابقة
- قائمة مستشارين باستخدام أيقونات فقط وليس صوراً حقيقية
- كل كرت يحتوي: الاسم الأول، التخصص، التقييم، وقت الرد المتوقع
- أزرار:
  - "ابدأ محادثة"
  - "احجز موعد"

5. **طلباتي**
- بطاقات الطلبات: رقم الطلب، التاريخ، الحالة، الإجمالي
- شريط حالة: جديد / جاري التجهيز / تم الشحن / تم التسليم
- زر داخل كل طلب: "عرض التفاصيل"
- زر إضافي: "إعادة الطلب"

6. **السلة والدفع**
- المنتجات المضافة
- تعديل الكمية
- حذف منتج
- ملخص السعر: subtotal + VAT + total
- أزرار الدفع: مدى / Apple Pay / تحويل بنكي
- زر أساسي كبير: "تأكيد الطلب"

7. **حسابي**
- بيانات مختصرة: الاسم الأول، المدينة، نوع المزرعة
- بطاقات: العناوين / طرق الدفع / اللغة / الدعم / تسجيل الخروج

**[5] 🎯 وظائف كل زر بشكل صريح**
- زر "اطلب الآن": يفتح شاشة السوق مباشرة
- زر "استشارة فورية": يفتح شاشة الاستشارات على تبويب "فورية"
- زر "تتبع شحنة": يفتح شاشة طلباتي مع تمرير تلقائي لآخر طلب نشط
- زر "إعادة طلب": ينسخ آخر طلب إلى السلة
- زر "أضف للسلة": يضيف المنتج ويظهر Toast "تمت الإضافة إلى السلة"
- زر "+" : يزيد الكمية بمقدار 1
- زر "-" : ينقص الكمية بمقدار 1، وإذا أصبحت 0 يظهر تأكيد حذف
- زر "استشر خبير عن هذا المنتج": يفتح محادثة جديدة مع تمرير اسم المنتج تلقائياً داخل الرسالة
- زر "ابدأ محادثة": يفتح شاشة محادثة مباشرة مع المستشار
- زر "احجز موعد": يفتح Bottom Sheet لاختيار اليوم والوقت
- زر "عرض التفاصيل": يفتح صفحة الطلب كاملة
- زر "إعادة الطلب": يضيف جميع عناصر الطلب السابق إلى السلة
- زر "حذف منتج": يزيل العنصر من السلة بعد نافذة تأكيد
- زر "تأكيد الطلب": يفتح شاشة اختيار الدفع ثم صفحة نجاح الطلب
- زر "الدعم": يفتح صفحة تواصل تحتوي واتساب، اتصال، ونموذج رسالة
- زر "تسجيل الخروج": يفتح Modal تأكيد

**[7] ✨ توجيهات الجودة**
- استخدم Auto Layout في كل العناصر
- اجعل كل شاشة مكتملة المحتوى وليست Placeholder
- لا تستخدم Lorem Ipsum
- استخدم أسماء أولى فقط: فهد، نورة، خالد، مها
- استخدم مدن سعودية: الرياض، القصيم، حائل، الأحساء
- جميع الأسعار بالريال السعودي: ر.س
- وفر حالات الضغط والانتقال والنجاح والخطأ لكل إجراء
